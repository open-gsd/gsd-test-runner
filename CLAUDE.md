Guidance for AI agents working in `open-gsd/gsd-test-runner`.

## Agent skills

### Issue tracker

Issues live in this repo's GitHub Issues (`open-gsd/gsd-test-runner`), accessed via the `gh` CLI. See `docs/agents/issue-tracker.md`.

### Triage labels

Canonical triage-role names are used as the actual label strings; all five state labels exist, plus org-convention companions (`confirmed-bug`, `approved-enhancement`, `approved-feature`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context — one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
